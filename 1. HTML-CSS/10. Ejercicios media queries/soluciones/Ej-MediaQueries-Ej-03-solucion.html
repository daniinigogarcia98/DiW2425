Ejercicio 3. Modifica el color de fondo de <body> a “pink” si la anchura del viewport es de 800px o mayor, a “lightgreen” si está entre 400 y 799px. Si es menor de 400px, color de fondo será “lightblue”.

¡Cuidado! La siguiente solución funciona, pero no respeta el principio de mobile first:

body {
  background-color: pink;
}

@media (max-width: 800px) {
  body {
    background-color: lightgreen;
  }
}

@media (max-width: 400px) {
  body {
    background-color: lightblue;
  }
}

SIEMPRE es mejor hacerlo mobile first. Es decir, establecer primero los estilos para pantallas pequeñas y añadir las media queries después para anchuras mayores. Por tanto, mejor así:

body {
  background-color: lightblue;
}

@media (min-width: 400px) {
  body {
    background-color: lightgreen;
  }
}

@media (min-width: 800px) {
  body {
    background-color: pink;
  }
}
