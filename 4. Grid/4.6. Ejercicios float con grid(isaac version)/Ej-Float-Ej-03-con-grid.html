Explicación:

Algunas formas de hacerlo con grid:

  1. Con áreas, sería poniendo en el contenedor grid-template-areas: "a c" "b c" y grid-template-columns: 1fr auto para que la primera columna se estire. Además de situar los divs con grid-area, tenemos que darle al Div2 justify-self: start y align-self: start para que no se estire ni en horizontal ni en vertical. También hay que darle a Div1 align-self: start para que no se estire en vertical (en horizontal no se estirará porque su columna no lo hace al tener el valor auto).

  2. Sin áreas, sería usando un grid de 2x2 con grid-template-columns: 1fr auto, donde Div1 ocupase la segunda columna completa, posicionándolo con grid-row: 1/3 y grid-column: 2/3. Como con la solución anterior, tenemos que darle al Div2 justify-self: start y align-self: start para que no se estire ni en horizontal ni en vertical.
