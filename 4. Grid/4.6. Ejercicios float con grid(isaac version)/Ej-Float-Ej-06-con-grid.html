La forma más fácil, rápida y eficaz de hacer este ejercicio con grid es simplemente poniendo en el contenedor grid-auto-flow: column. No hace falta usar grid-template-columns para definir las columnas porque serán auto por defecto. La primera columna ocupará los 200px establecidos para la imagen y la segunda columna se estirará para rellenar la anchura restante. Solo faltaría align-self: start; en el div con texto para que no se estire verticalmente.
