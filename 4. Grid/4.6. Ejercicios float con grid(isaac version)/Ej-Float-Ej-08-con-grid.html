Explicación:

Para este ejercicio tenemos varias posibles soluciones, que serían poner en el contenedor alguna de estas instrucciones:

  1. grid-auto-flow: column. Esta forma sólo funciona si todas las imágenes tienen el mismo tamaño y el texto también es el mismo. Vamos, que no.

  2. grid-template-columns: repeat(3, auto). Tiene el mismo problema que la instrucción anterior porque en ambas las columnas se definen con un tamaño auto.

  3. grid-template-columns: repeat(3, 33.33%). Esta solución es buena, pero falla si se utiliza column-gap, dado que se produciría desbordamiento horizontal en el contenedor.

  4. grid-template-columns: repeat(3, 1fr). Esta es una buena solución. Funciona con tamaños de imágenes distintos, textos distintos y column-gap no produce desbordamiento.

  5. grid-template-columns: repeat(auto-fit, minmax(50px, 1fr)). Análoga a la anterior, pero incluso mejor porque no necesitamos definir el número de columnas.
