Explicación:

Con el header de este ejercicio hay que tener en cuenta que Navigation ocupa lo mismo que sub content y sub content 2. Para conseguir ese efecto simplemente tenemos que establecer el mismo tamaño de columnas en ambos sitios, por ejemplo con grid-template-columns: 25fr 70fr o, mejor, grid-template-columns: 1fr 3fr. Para la separación podemos usar column-gap. Además, tenemos que poner justify-self: start en el logo para que no se estire en horizontal.

En la parte central, con establecer las columnas y la separación como en el header basta para conseguir el layout pedido.
